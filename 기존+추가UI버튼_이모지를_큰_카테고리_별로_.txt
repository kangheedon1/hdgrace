### **라이브 스트리밍 관리**
- 🟢 LIVE 고정 시청자 유지
- 🟣 Shorts 시청 최적화

- 🛡️ 지메일 보안 설정
- 📧 지메일 복구 이메일 설정

### **유튜브 관리**

- 📝 유튜브 채널 설명 설정
- 🏷️ 유튜브 채널 태그 설정

### **자동화 기능**
- 🤖 자동 댓글봇
- 🔔 자동 구독 관리
- ⏰ 스케줄된 시청 시간

### **보안 및 안전**

- 🛡️ 탐지 방지 모드
- 🔍 보안 상태 모니터링
- ⚠️ 예외 처리 설정

### **다국어 및 지역**

- 🌍 지역별 콘텐츠 최적화
- 🗣️ 언어별 시청 패턴

### **프록시 관리**
- 📡 IMEI + 회전 프록시 설정
- 📶 고정 프록시 및 Google 계정 없이

- 모든 기능은 UI 버튼을 통해 선택하여 실행 가능

- 시스템 상태는 상단 상태 표시줄에서 실시간으로 확인 가능
- 실행 버튼을 통해 모든 선택된 기능을 한 번에 시작할 수 있음
